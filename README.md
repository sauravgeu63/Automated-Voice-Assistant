# Automated-Voice-Assistant
This is a python based fully Automated voice assistant that can make your work easier , you can just give your voice command to open many tasks like Emails, WhatsApp ,know the date and time open any browsers and search anything on Youtube, For UI I used Tkinter GUI.

** About Project** 

1. You need to import all the libraries correctly to operate this project.
2. All the files must be  in a single folder.
3. Use pip command to get the Libraries (Easier for You).
4. Use VS code (Suggested).
5. After clicking Run python file , a GUI Appear With multiple Buttons.
6. Click on the Speak button to give commands.
7. You canb also give commands by clicking on the specific task Buttons.
